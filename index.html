Stuff you may need to know

  1. Assets are ~2Mb if there is a brief pause at start
  2. This is javascript and webgl based so those need to be enabled
  3. Get to end of the map(s) by collecting fuel from planet encounters or hitting an enemy with a rocket.
  4. Moving the ship or firing the ships weapon costs fuel as well as damage taken to the shield from enemy fire.
  5. Levels get progressively more difficult so balance is needed.
